Lightweight 3D modeling and simulation framework

easy-mode modeling of space, planets, satellites, whatever. in the browser. no plugins.

Supported Browsers

universe.js works everywhere WebGL does

Want to know if your browser supports WebGL? Check out doesmybrowsersupportwebgl.com

Easy to get started

universe.addObject({
	id : 'hotness',
	update : function(elapsedTime) {
		// move and rotate object
		object.position.x += 10;
		object.rotation.y += 0.1
	},
	draw : function() {
		universe.draw(this.id, object, isScaled);
	}
});
				
universe.setCurrentUniverseTime(time);
				
universe.play();
